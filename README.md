# Todo App (Управление задачами)

Веб-приложение для управления списком задач с полным CRUD-функционалом. Реализовано на чистом JavaScript с использованием JSON Server в качестве бэкенда.

## Реализованные функции (доработки)
- Добавление, просмотр, редактирование и удаление задач
- Отметка задачи как выполненной / активной
- Дата и время создания задачи
- Inline-редактирование (по двойному клику на заголовке)
- Фильтрация: Все / Активные / Выполненные
- Счётчик активных задач
- Очистка всех выполненных задач одной кнопкой
- Хранение данных в JSON-файле через JSON Server

## 🛠 Стек технологий
- Frontend: HTML, CSS (Bootstrap), JavaScript (ES6+)
- Backend: JSON Server (имитация REST API)
- База данных JSON-файл (db.json)
